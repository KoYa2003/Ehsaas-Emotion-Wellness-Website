# Ehsaas 

Ehsaas is a full-stack mental wellness website that allows users to record, track, and reflect on their emotional states to promote emotional awareness and self-care.

---

## Problem Statement

Many individuals struggle to understand and manage their emotional health. Ehsaas provides a simple and supportive digital platform for users to express their feelings, observe mood patterns, and receive wellness-oriented guidance.

---

## Features Implemented

- Text-based emotion input system
- Mood visualization dashboard
- Rule-based wellness recommendation system
- User mood history tracking

---

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express
- **Database:** MySQL

---

## Current Status

This project is under active development. Future improvements include:

- AI-based emotion classification using NLP
- Personalized therapy chatbot
- Mobile-friendly responsive version

---

## Why this project?

Mental health awareness is often overlooked in daily life. Ehsaas was created to encourage emotional self-reflection and provide a safe, private, and easy-to-use platform for understanding personal emotional wellbeing.
